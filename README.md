# Shankh Cabs Pune

A professional, responsive, and modern website for Shankh Cabs Pune, featuring real-time booking links, fleet showcase with high-quality images, and multi-location map integration.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Fleet Showcase**: High-quality images for Hatchback, Sedan, SUV, and Traveller categories.
- **Multi-Location Maps**: Integrated Google Maps for Hinjawadi and Walvekar Nagar offices.
- **One-Click Booking**: Easy access to "Call Now" and WhatsApp booking buttons.
- **Trust Badges**: Highlighted key benefits like Verified Drivers and 24/7 Service.
- **SEO Optimized**: Professional meta tags and descriptions for better search visibility.

## Tech Stack

- **React**: Core UI library.
- **Vite**: Modern frontend build tool.
- **SASS/SCSS**: Modular and maintainable styling.
- **Lucide Icons**: Clean and modern iconography.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```
